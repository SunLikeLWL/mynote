# 为什么使用虚拟DOM



### 为什么使用Virtual DOM

1、手动操作DOM比较麻烦，还需要考虑浏览器兼容性问题，虽然有jQuery等库实现了简化DOM操作，
   但是随着项目的复杂，DOM操作复杂提升

2、为了简化DOM的复杂操作出了MVVM框架，MVVM框架解决了视图和状态的同步问题

3、为了简化视图的操作我们可以使用模板引擎，但是模板引擎没有解决跟踪状态变化的问题，
   于是Virtual DOM出现了

4、Virtual DOM的好处是当状态改变时不需要立即更新DOM，只需要一个虚拟的树来描述DOM，Virtual DOM内部 
   将弄清楚如何有效（diff）更新DOM
