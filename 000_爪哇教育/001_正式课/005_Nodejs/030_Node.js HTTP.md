# Node.js HTTP



### OSI模型

1、应用层
各种应用程序协议


2、表示层
数据的格式化，数据加密解密，数据的压缩解压缩

3、会话层
简历、管理、终止实体之间的绘画连接

4、传输层
数据的分段及重组；提供端到端的数据服务


5、网络层
将分组从源端传送到目的地的端：逻辑地址；路由选择

6、数据链路层
将分组数据封装成帧；实现两个相邻节点之间的通信；差错检测


7、物理层
在介质上传输比特；提供机械的和电气的规约
